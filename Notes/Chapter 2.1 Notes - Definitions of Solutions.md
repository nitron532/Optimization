
Global minimizers are hard to find, so we often settle for local minima.

Recall that a neighborhood of $x^*$ is simply an open set that contains $x^*$

A point $x^*$ is a global minimizer if $\forall x, f(x^*) \leq f(x)$

A point $x^*$ is a weak local minimizer if $\exists$ neighborhood $N$ such that $\forall x \in N, f(x^*) \leq f(x)$ .
A strict/strong local minimizer restricts the inequality to < only, and $x \neq x^*$

A point $x^*$ is an isolated local minimizer if it is the only local minimizer in its neighborhood.
